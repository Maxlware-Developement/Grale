# Grale

Grale est un navigateur internet développé par Maxlware DVT pour but de faire marcher sur tout type d'ordinateur. Le code vient innitiallement de [Mxlw Browser](https://github.com/Maxlware-Developement/Mxlw-browser) mais avec une refonte graphique et d'autres fonctionnalités.

> [!WARNING]
> Cette version est en test et en mode dévellopement seulement. Pour avoir une version LTS stable nous vous conseillons d'aller sur le repo de [Mxlw Browser sur la version *1.1.1 LTS-S-G*](https://github.com/Maxlware-Developement/Mxlw-browser/releases/tag/LTS). Pour plus d'informations vous pouvez aller regarder le [*README* de Mxlw Browser](https://github.com/Maxlware-Developement/Mxlw-browser/blob/main/README.md).

---

## Lancer en mode développement
0: Les prérequis sont *Nodejs 24.11.1*, *python ou python3*<br>
1: Télécharger le repo
```
git clone https://github.com/Maxlware-Developement/Grale.git
```
2: Aller dans le dossier
```
cd C:/{gngngn}/Grale/src
```
3: Télécharger les libraries
```
npm install
```
4: Lancer
```
npm start
```
